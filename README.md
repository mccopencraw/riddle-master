# 🎮 Riddle Master - English Brain Teasers

An interactive English riddle game for primary school students with sticker rewards!

## 🎯 Features

- **Easy Level**: 4-choice questions with pictures
- **Medium Level**: Unscramble letters to spell the answer
- **Sticker Collection**: Earn stickers for completing the game
- **Focus Reminder**: Encourages players to complete the full game
- **Audio Support**: Riddles are read aloud (TTS)

## 🎨 Game Flow

1. Choose level (Easy/Medium)
2. Answer 10 riddles
3. Earn score based on correct answers
4. Collect stickers in the Store
5. Decorate your profile with stickers

## 🏆 Scoring System

| Score | Rating | Stickers |
|-------|--------|----------|
| 100 | 🌟 Legendary | 5 |
| 80-99 | ⭐ Excellent | 4 |
| 60-79 | 👍 Good Job | 3 |
| 40-59 | 💪 Keep Trying | 2 |
| 20-39 | 🌱 Nice Try | 1 |
